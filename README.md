# AporiaLab Frontend 🧠

منصة نقاشات عربية — حيث تلتقي الأفكار

## 🚀 النشر على Vercel

هذا المشروع مُعدّ للنشر المباشر على Vercel مع GitHub.

### المتطلبات:
- حساب GitHub
- حساب Vercel
- Backend منشور (aporialab-backend)

### خطوات النشر:

1. رفع الكود إلى GitHub Repository
2. ربط Repository بـ Vercel
3. إضافة متغيّر البيئة:
   - `VITE_API_URL` = رابط Backend (مثلاً: `https://zaingarad07-hue-aporialab-backend.vercel.app`)
4. Deploy

## 🛠️ التقنيات المستخدمة

- **React 19** + **TypeScript**
- **Vite** للبناء السريع
- **Tailwind CSS** + **shadcn/ui** للتصميم
- **React Router** للتنقل
- **i18next** للترجمة (عربي/إنجليزي)

## 📂 هيكل المشروع

```
src/
├── components/     # مكونات UI
├── context/        # AuthContext
├── i18n/          # ملفات الترجمة
├── pages/         # الصفحات
├── sections/      # أقسام الموقع
└── services/      # api.ts
```
